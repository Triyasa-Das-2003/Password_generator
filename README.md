## ‚ú® Features

* **Secure Password Generation:** Creates 8-character passwords that include a mix of uppercase letters, lowercase letters, numbers, and symbols.
* **Guaranteed Complexity:** The generation script ensures at least one of each character type (uppercase, lowercase, number, and symbol) is included in every password.
* **One-Click Copy:** Easily copy the generated password to the clipboard using the copy icon.
* **Visual Feedback:** Displays a "copied" message confirmation when the password is copied.
* **Responsive Design:** The layout is functional and looks great on different screen sizes.
